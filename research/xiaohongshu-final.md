### 第5-7张：解决方案

**第5张：理想决策流程（上半部分）**

**PlantUML源码**：
```plantuml
@startuml
|媳妇|
start
:提出需求;
note right : "孩子6个月了，该加辅食"

|婆婆|
:表达意见;
note right : "我当年是4个月开始"

|丈夫|
:收集信息;
note right : 查医生建议\n查育儿知识\n听双方理由

|家庭会议|
:共同讨论;
note right : 所有人发言\n不打断

if (达成一致？) then (是)
  |全员|
  :执行决策;
else (否)
  |丈夫|
  :作为决策者;
  :说明理由;
endif
stop
@enduml
```

**配文**：
```
这样决策，家庭和谐度提升80%

1️⃣ 媳妇提出需求
```
