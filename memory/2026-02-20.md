#### 2. 编程语言选择的理性化
**来源**：Ginger Bill（Odin语言作者）

**核心观点**：
- 语法不等于语义，语义才是语言本质
- 许多人混淆语法和语义，仅凭"好看程度"选择语言
- 第一接触偏见影响语言选择

**声明语法的三种类型**：
1. 类型优先：`type name = value`（如Rust、TypeScript）
2. 名称优先：`name: type = value`（如Odin、Pascal）
3. 限定符优先：`var name type = value`（如Go、C#）

**技术细节**：
- Odin采用"名称优先"语法，配合强大的类型系统
- 分号策略：现代语言（Go/Python/Odin）通过ASI处理可选分号
- 类型设计：Odin的"类型在左，使用在右"更易理解

#### 3. Weathr - 终端天气应用
**来源**：GitHub项目

**技术栈**：
- 语言：Rust
- 天气数据：Open-Meteo（免费API，无需密钥）
- 渲染：ASCII艺术 + 真彩色终端

**核心功能**：
- 实时天气动画（雨、雪、雷暴、日夜循环）
- 自动位置检测（ipinfo.io）
- 配置系统（TOML格式）
- 单位转换（公制/英制）

**技术价值**：
- 展示Rust在CLI开发的优势
- ASCII艺术的现代应用
- 零依赖API降低使用门槛
- 无障碍考虑（NO_COLOR支持）
