## 纯 URL 链接格式修复 (07:45)

### 问题描述
用户反馈：投资大师思想精华有类似 `https://nanochat.karpathy.ai` 的格式问题

### 根本原因
`markdown-to-notion.py` 的 `parse_inline_formatting` 函数只处理：
- ✅ 粗体 `**text**`
- ✅ 斜体 `*text*`
- ✅ 代码 `` `code` ``
- ❌ **纯 URL** `https://...`（缺失）

导致以下格式不被正确解析：
```markdown
**链接：** https://example.com
- **GitHub**: https://github.com/...
```

### 修复措施
1. 在 `parse_inline_formatting` 函数中添加纯 URL 模式：
   ```python
   (r'(https?://[^\s\)]+)', {"link": True})
   ```

2. 在匹配逻辑中添加处理：
   ```python
   elif matched.startswith('http'):
       result.append({
           "type": "text",
           "text": {"content": matched, "link": {"url": matched}}
       })
   ```
