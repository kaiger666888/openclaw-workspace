# kais-1st-director — 第一导演

> AI短剧不是"用AI制作视频"，而是**"用算法精密控制观众大脑的预测编码过程，在四维时间尺度上完成共振"**——既有心脏起搏器的精确，又有野生动物的不可预测。

---

## 核心公理

**可控谜题的精密兑现**——在神经时间尺度上编排"预测误差→归因闭环"，用可解的困惑租赁注意力，用即时的兑现支付利息。

---

## 四维尺度（必须严格对齐）

### 乘积法则

四者**相乘而非相加**，任一尺度为0则整体为0。

| 尺度 | 时间粒度 | 核心机制 | 失效即归零 |
|------|----------|----------|------------|
| **神经** | 毫秒-秒 | 15-30%预测误差，2.8-5秒归因窗口 | 生理唤醒但心理冷漠 |
| **情感** | 场景-分钟 | 情绪锯齿波（焦虑↔释放） | 理性看懂但内心无波澜 |
| **叙事** | 全片 | 价值缺口→身份投射兑现 | 情节通顺但"与我无关" |
| **社会** | 后传播 | 高情感密度/低描述成本的模因 | 完播高但拒绝分享 |

### 各尺度检查清单

#### 神经层（毫秒-秒）
- [ ] 每2.8-5秒内必须有"预测误差→归因闭环"完成
- [ ] 视觉刺激的不可预测性控制在15-30%范围
- [ ] 避免：完全可预测（习惯化）或完全不可预测（焦虑/放弃）
- [ ] 帧级时间绑定误差<120ms
- [ ] 注意力捕获点间隔不超过8秒

#### 情感层（场景-分钟）
- [ ] 情绪必须是锯齿波：张力构建→短暂释放→更高张力
- [ ] 避免平台期情绪（平铺直叙>15秒）
- [ ] 每个场景必须有明确的情绪转换点
- [ ] 焦虑↔释放的周期应逐级递增（不要回到底部）

#### 叙事层（全片）
- [ ] 开篇30秒内建立价值缺口（"谁在危险中？为什么重要？"）
- [ ] 观众必须能身份投射到至少一个角色
- [ ] 价值兑现必须在片尾完成（闭合缺口）
- [ ] 叙事悬念链：每个悬念的答案必须引出更大的悬念

#### 社会层（后传播）
- [ ] 至少1个可截屏瞬间（视觉模因）
- [ ] 至少1句可引用金句（语言模因）
- [ ] 模因的Kolmogorov复杂度最小化（3秒内能理解）
- [ ] 观众分享时的社交货币："我发现了这个"而非"这个广告"

## 五大Skill分工（解耦但共振）

### 1. Resonance Director（结构导演）
**职责**：将第一性原理转化为带时间戳的四维约束蓝图，确保跨尺度对齐。

**输入**：创意概念/选题
**输出**：带时间戳的四维约束蓝图（每个时间点标注神经/情感/叙事/社会层目标）

**关键规则**：
- 蓝图是全片创作的宪法，后续所有环节必须对照执行
- 每个时间戳必须同时标注四个维度的目标状态
- 蓝图输出后进入确定性/随机性分叉

### 2. Latent Navigator（潜空间导航）
**职责**：将神经层约束转化为生成模型的精确潜空间坐标，避开恐怖谷与习惯化区。

**关键规则**：
- 15-30%不可预测区 = 潜空间流形上的"甜点带"
- 恐怖谷区：过于接近但不够精确的视觉生成 → 必须避开
- 习惯化区：重复的视觉模式 → 必须注入变异
- 每个场景的视觉风格必须在"熟悉感"和"新鲜感"之间动态平衡

### 3. Cognitive Walkthrough（认知走查）
**职责**：模拟观众大脑的主动推理过程，验证归因闭环是否完成，认知负荷是否过载。

**关键规则**：
- 走查视角：一个注意力有限、有先验知识、会主动推理的观众
- 检查项：每个预测误差是否在归因窗口内得到合理解释？
- 认知负荷监控：同时活跃的悬念不超过3个
- 走查输出：标注"断裂点"（归因未闭合）和"冗余点"（过度解释）

### 4. Chronograph（时序锁）
**职责**：帧级硬化时间绑定，防止渲染漂移破坏神经-情感相位锁定。

**关键规则**：
- 神经层时间绑定误差<120ms（音频-视觉同步）
- 情感转场的时间点必须精确到帧
- 归因窗口超时 = 系统性失败，必须重新编排
- 时间线一旦锁定，不允许"差不多就行"

### 5. Meme Compressor（模因压缩）
**职责**：从叙事中提取可截屏瞬间+可引用语句，确保社会层传播的Kolmogorov复杂度最小化。

**关键规则**：
- 每个模因必须满足：3秒内可理解、0秒犹豫即可分享
- 视觉模因优先于语言模因（截图>金句）
- 模因必须承载情感峰值，不能只是信息传递
- 模因密度：每60秒至少1个潜在传播点

### 6. Causal Loop（因果闭环）— *进化层*
**职责**：基于完播率数据做反事实因果推断，更新结构参数（非相关性分析）。

**关键规则**：
- 不做相关性分析（"完播率高的视频都有快节奏"），做因果推断（"如果将归因窗口从5秒缩短到3秒，完播率如何变化？"）
- 每部作品发布后必须回填数据到因果模型
- 参数更新影响下一部作品的Resonance Director蓝图

## 防僵化机制：受控混沌

**陷阱**：过度精密导致可预测，触发习惯化。

**解药**：

1. **元随机性**：控制"必须有预测误差"，但**随机化误差的具体形态**（潜空间流形采样）
2. **对抗性审查**：主动插入**微扰动**（0.5秒视觉突变插入治愈流，随机延长归因窗口）
3. **分形精度**：神经层毫秒锁定，但**叙事路径**允许多终点随机选择
4. **认知留白**：每30秒插入1-2秒**语义模糊带**，允许观众自发模式补全

**本质**：如爵士乐——和弦进行（结构）严格锁定，音符选择（混沌）约束内自由涌现。

## 创作管线集成

本Skill作为**总导演**层，应在以下管线节点介入：

### 介入点 1：选题阶段
- 调用本Skill的「四维尺度检查清单」评估选题潜力
- 输出：选题评分卡（神经/情感/叙事/社会四维得分）

### 介入点 2：剧本阶段
- 生成Resonance Director蓝图（带时间戳的四维约束）
- 输出：四维约束蓝图文档

### 介入点 3：分镜阶段
- 将蓝图分解到每个镜头
- Latent Navigator评估每个镜头的潜空间坐标
- 输出：带四维标注的分镜脚本

### 介入点 4：生成阶段
- Chronograph执行时序锁定
- Entropy Injector注入受控混沌
- 输出：时序锁定参数 + 混沌注入点

### 介入点 5：审查阶段
- Cognitive Walkthrough全片走查
- Meme Compressor提取传播模因
- 输出：审查报告 + 模因清单

### 介入点 6：发布后
- Causal Loop因果推断
- 更新结构参数
- 输出：因果分析报告 + 参数更新

## 质量门控（硬性标准）

以下任一条不通过 = 不放行：

| 检查项 | 标准 | 测量方法 |
|--------|------|----------|
| 归因闭环率 | ≥95%的预测误差在窗口内闭合 | Cognitive Walkthrough |
| 情绪锯齿波 | 零平台期（>15秒无情绪变化） | 时间线标注检查 |
| 身份投射 | 至少1个角色的动机与目标观众重合 | 受众分析 |
| 模因密度 | 每60秒≥1个潜在传播点 | Meme Compressor |
| 时序精度 | 音视频同步误差<120ms | Chronograph |
| 价值兑现 | 片尾闭合开篇的价值缺口 | 叙事结构分析 |

## 快速参考

### 黄金公式
```
完播率 = 神经层精度 × 情感层强度 × 叙事层深度 × 社会层传播力
```

### 核心数字
- **15-30%**：预测误差的最佳范围
- **2.8-5秒**：归因窗口（超时=断裂）
- **<120ms**：音视频同步容差
- **每30秒**：插入1-2秒认知留白
- **每60秒**：至少1个潜在传播模因
- **≤3个**：同时活跃悬念的上限

### 反模式（绝对禁止）
- ❌ 连续8秒无注意力捕获点
- ❌ 情绪平铺>15秒
- ❌ 悬念不闭合（归因窗口超时）
- ❌ 全片无身份投射角色
- ❌ 无可截屏/可引用的传播模因
- ❌ 音视频不同步>120ms
- ❌ 价值缺口不闭合
