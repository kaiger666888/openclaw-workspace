## 质量标准

### ✅ 必须包含
- 技巧必须有**具体操作步骤**
- 案例必须有**实际代码/命令**
- 资源必须有**来源链接**
- **至少 3 条**有价值的内容

### ❌ 不要包含
- 没有验证的技巧
- 过于基础的入门内容
- 与 Claude Code 无关的内容
- 纯营销性质的推广

### 📊 数量控制
- 今日技巧: 2-3 条
- 实战案例: 1-2 条
- 学习资源: 1-2 条
- **总计 4-8 条**，质量优先

---

## 评估标准

1. **实用性** - 能直接应用到开发中
2. **原创性** - 不是随处可见的基础内容
3. **可验证** - 技巧可以实际验证
4. **时效性** - 符合 Claude Code 最新版本
