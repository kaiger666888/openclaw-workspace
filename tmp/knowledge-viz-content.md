## 🛠️ 工具与库更新

### Mermaid.js
- **文本到图表的革命**: Markdown式语法，消除文档腐化问题
- **Claude集成**: 新Skill支持自动生成流程图、时序图、类图、状态图等
- **LobeHub Skills**: `docs-with-mermaid` 自动生成结构化文档并嵌入Mermaid图表

### Excalidraw
- **AI Agent Skill**: 支持实时画布操作、批量创建、截图导出
- **Obsidian集成**: 可编辑源文件（非截图），支持版本控制
- **文本转图表**: 自动分析内容识别概念、关系和流程

### 可视化三件套（Obsidian生态）

- **Excalidraw**: 手绘风格可编辑图表

### D3.js
- 依然是最强大的数据可视化库（107,000+ GitHub stars）
- 适用于新闻机构、金融分析、科研可视化

## 💡 设计理念

### 好的可视化原则
- **清晰 > 炫酷**: 信息传达优先
- **渐进披露**: 从概览到细节
- **上下文完整**: 标题、标签、图例不可或缺
- **色彩克制**: 语义化用色，避免彩虹配色

### 工具选择指南
| 需求 | 推荐工具 |
|------|----------|
| 快速流程图 | Mermaid + Claude |
| 手绘风格/头脑风暴 | Excalidraw |
| 复杂数据分析 | D3.js / Observable |
| 知识管理 | Obsidian Canvas |
| 企业仪表盘 | ThoughtSpot / Tableau |
