1. 访问 http://www.plantuml.com/plantuml
2. 粘贴 puml 文件内容
3. 点击提交
4. 下载生成的图片

**方法2：本地渲染**
```bash
# 安装 PlantUML
brew install plantuml

# 渲染图片
plantuml mother-in-law-system.puml
plantuml mother-in-law-conflict.puml
plantuml mother-in-law-decision.puml
```
