## 即梦(Jimeng)浏览器登录流程 (21:20)
- **成功方法**: Playwright non-persistent context + `--start-maximized` + background 保持
- **关键步骤**:
  1. 打开 `https://jimeng.jianying.com/ai-tool/home`
  2. 点击「图片生成」卡片 (`.button-YBvLch` filter hasText)
  3. 弹出协议弹窗 → 点击「同意」按钮 (`.lv-modal-content button` filter hasText:'同意' last())
  4. 点击登录按钮 (`[class*=login-button]` force click)
  5. 等待新 tab 弹出 (`open.douyin.com/platform/oauth/pc/auth`) → 二维码页面
  6. 截图发送给用户扫码
- **注意事项**:
  - CSS class 是 CSS modules hash，每次不同，不能用固定 class，用 text filter
  - 必须用 `background: true` + `setInterval` 保持浏览器进程不退出
  - 协议弹窗没有 checkbox，只有「不同意」和「同意」两个按钮
  - 登录按钮被弹窗遮挡时需要 force click
