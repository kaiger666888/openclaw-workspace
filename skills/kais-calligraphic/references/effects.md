# 浮现效果技术说明

## 1. horizontal_wipe（横向擦除）

从左到右逐列显示文字，模拟毛笔从左向右书写。

**实现**：生成与文字同尺寸的 mask，每帧将 mask 的右边界向右移动。边缘添加微小随机偏移，模拟毛笔的不均匀笔触。

**适用**：横排文字，最经典的书法浮现效果。

## 2. vertical_wipe（纵向擦除）

从上到下逐行显示文字。

**实现**：与 horizontal_wipe 类似，但方向改为从上到下。

**适用**：竖排文字。

## 3. fade_in（渐变浮现）

文字整体透明度从 0 渐变到 255。

**实现**：每帧生成全图同值的 mask，值从 0 线性增长到 255。

**适用**：需要柔和效果的场景。

## 4. ink_spread（水墨散开）

模拟墨滴落在宣纸上向四周散开的效果。

**实现**：
1. 在文字区域均匀分布多个"墨滴中心"
2. 每个墨滴从中心向外以高斯衰减扩散
3. 所有墨滴取最大值合并
4. 用 mask 裁切到文字形状

**适用**：强调艺术感、古典氛围的视频。

## 5. ink_bleed（墨迹渗透）

模拟墨汁渗透宣纸的效果，从文字中心区域开始向外扩散。

**实现**：
1. 计算每个像素到文字中心的归一化距离
2. 用 threshold 函数控制渗透前沿
3. 添加高斯噪声模拟纸张纤维的不均匀渗透

**适用**：需要独特视觉效果的场景。
