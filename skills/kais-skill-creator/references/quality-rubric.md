# 质量评分标准详解

## 🎯 触发精确度（20分）

| 扣分项 | 分值 | 说明 |
|--------|------|------|
| description 为空或过短 | -5 | 无法有效触发 |
| 触发词少于 2 个 | -3 | 覆盖面不足 |
| 缺少中文触发词 | -3 | 忽略中文用户 |
| 缺少具体触发场景 | -4 | AI 无法判断何时使用 |

## 📐 结构完整性（20分）

| 扣分项 | 分值 | 说明 |
|--------|------|------|
| SKILL.md 不存在 | -20 | 致命 |
| 缺少 frontmatter | -5 | 无法识别 |
| 缺少 name | -3 | 无法引用 |
| 缺少 description | -3 | 无法触发 |
| 章节少于 2 个 | -5 | 内容不完整 |
| 脚本无执行权限 | -2 | 无法运行 |
| 引用路径不存在 | -2/个 | 链接断裂 |

## ⚡ Token 效率（20分）

| 扣分项 | 分值 | 说明 |
|--------|------|------|
| > 500 行 | -5 | 超过建议值 |
| > 800 行 | -10 | 严重超标 |
| 大段代码块(>20行) | -3 | 应移到 scripts/ |
| references/ 为空 | -3 | 未利用渐进式披露 |

## 🔧 可执行性（20分）

| 扣分项 | 分值 | 说明 |
|--------|------|------|
| Shell 语法错误 | -5/个 | 无法运行 |
| Python 语法错误 | -5/个 | 无法运行 |
| 缺少 shebang | -2 | 兼容性问题 |

## 🌏 本地化质量（20分）

| 扣分项 | 分值 | 说明 |
|--------|------|------|
| description 无中文 | -3 | 忽略中文用户 |
| 中文比例 < 20% | -5 | 未本地化 |
| 技术术语无中文对照 | -2 | 阅读体验差 |
