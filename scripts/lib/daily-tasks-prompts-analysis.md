# 每日任务 Prompt 分析与改进

## 任务目标与 Prompt 评估

### 1. 每日英语德语 (04:10)
**目标**: 每天学习实用的英语和德语知识点，形成可回顾的学习笔记

**当前 Prompt 问题**:
- ❌ 没有明确学习内容的难度级别
- ❌ 没有要求使用颜色标注重点
- ❌ 语法要点"多样化"太模糊
- ❌ 没有明确内容结构

**改进方向**:
```markdown
目标用户: 中级学习者，希望提升实际应用能力
内容结构:
  - 📚 英语部分
    - 语法要点（用蓝色标注术语）
    - 5个高级词汇（含音标、例句）
    - 每日一句（名言或实用句）
  - 📚 德语部分
    - 语法要点（用蓝色标注术语）
    - 动词变位表（表格形式）
    - 5个实用词汇
  - ✅ 学习检查清单
格式要求: 使用 callout 高亮重点，颜色标注关键术语
```

### 2. AIGC前沿总结 (04:20)
**目标**: 快速了解 AI 领域最新动态，把握行业趋势

**当前 Prompt 问题**:
- ❌ 搜索范围太广，容易超时
- ❌ 没有明确要对比的数据维度
- ❌ 没有要求总结趋势

**改进方向**:
```markdown
目标用户: 关注 AI 行业的技术人员
核心内容:
  - 🔥 今日头条（1-2条最重要新闻）
  - 📊 模型动态（新模型发布、性能对比）
  - 💰 商业动态（融资、收购、产品发布）
  - 🔧 工具更新（新工具、新功能）
  - 📈 趋势观察（行业分析）
格式要求:
  - 用红色标注负面/下降数据
  - 用绿色标注增长/正面数据
  - 用 callout 总结核心观点
```

### 3. 每日新闻 (04:30)
**目标**: 快速获取科技领域重要新闻，节省信息筛选时间

**当前 Prompt 问题**:
- ❌ "至少10条"太多，质量难保证
- ❌ 没有优先级排序
- ❌ 没有要求筛选标准

**改进方向**:
```markdown
目标用户: 技术从业者，需要快速了解行业动态
筛选标准:
  - 影响力: 影响多少人？
  - 相关性: 与技术/编程/创业的相关程度
  - 时效性: 是否今日发生？
内容结构:
  - 📰 Top 3 头条（用红色标注重要性）
  - 🔧 技术/开发工具
  - 💼 商业/创投
  - 🛡️ 安全/隐私
  - 📊 每条新闻: 标题 + 1-2句摘要 + 来源链接
格式要求: 控制在 8-10 条，质量优先于数量
```

## Prompt 改进原则

### 1. 明确目标用户
每个任务都应该明确"谁在看这个笔记"

### 2. 结构化内容
不要让 AI 自由发挥，给出明确的内容结构

### 3. 可操作性
内容应该是"可以做什么"，而不是"知道什么"

### 4. 格式规范
明确使用哪些 Notion 块类型和颜色

### 5. 质量标准
明确"好的输出"是什么样的
