# kais-story-outline - 故事大纲创作技能

> 世界顶级幽默/讽刺/情感作品创作技法 × 专业大纲结构 × 时事热点融入

## 激活条件

当用户提到以下关键词时激活：
- "写个故事"、"故事大纲"、"剧本大纲"、"创作大纲"
- "讽刺故事"、"幽默故事"、"搞笑故事"
- "情感故事"、"治愈故事"、"催泪故事"
- "短视频脚本"、"单元剧大纲"、"系列剧大纲"
- "帮我编个故事"、"创作一个故事"

## 核心流程

```
创作前调研 → 确定风格/格式 → 大纲生成 → 质量自检 → 交付
```

---

## Step 1: 创作前调研（可选但推荐）

如果用户没有指定具体题材，用 `web_search` 搜索当前时事热点：

```
搜索关键词："今日热点 社会话题 热门讨论" + 相关领域
```

从热点中提取：
- **深层焦虑**：这个热点触发了人们什么深层恐惧/渴望？
- **讽刺靶点**：什么行为/现象值得被放大和嘲讽？
- **情感锚点**：什么情感是人们最渴望被触动的？

## Step 3: 大纲生成

### 幽默/讽刺模块

#### 核心原则
1. **荒诞的逻辑自洽**：设定可以离谱，但必须被严肃对待
2. **意外感 > 工整感**：人类创作中那些"不太对劲"的地方最有魅力
3. **克制留白**：说70%比说100%更有力，让读者自己发现笑点
4. **观察 > 技巧**：最好的讽刺来自对生活的真实观察

#### 讽刺手法选择

**根据目标选择手法：**

| 手法 | 定义 | 使用场景 | 案例 |
|------|------|---------|------|
| 反差讽刺 | 制造两个极端的反差 | 社会讽刺、政治讽刺 | 《黑镜》：科技便利 vs 人性代价 |
| 荒诞逻辑 | 荒谬前提下的严密逻辑链 | 科幻讽刺、寓言 | Douglas Adams：因修路毁灭地球 |
| 日常陌生化 | 把熟悉的事物变陌生 | 揭示习以为常的荒谬 | 《世界奇妙物语》：发现自己是配角 |
| 礼貌的残酷 | 形式克制 vs 内容犀利 | 职场、政治讽刺 | 《是，大臣》：最得体的话说最残忍的事 |
| 欧·亨利反转 | 结尾完全颠覆预期 | 短篇、单元剧 | K-J.帕克：结尾轻轻一句颠覆全篇预期 |
| 重复与变异 | 同一模式重复出现，每次略有不同 | 系列作品 | 《银河系漫游指南》："别慌"和毛巾的反复出现 |

#### 避免"AI味"的自检

| ❌ AI味特征 | ✅ 人类创作者做法 |
|------------|------------------|
| 用"然而"制造反转 | 用场景转换、对话打断、突然沉默来制造意外 |
| 讽刺对象太明显 | 讽刺大家习以为常但从未质疑的日常 |
| 笑点有预告感 | 笑点藏在看似正常的叙述中 |
| 结构完美对称 | 刻意留一些"不太舒服"的不对称 |
| 万物皆可讽刺 | 知道什么时候不讽刺——真诚比讽刺更有力量 |

### 情感模块

#### 核心原则
1. **具体 > 抽象**：用一道菜、一句话、一个动作代替"他很想念你"
2. **行动 > 语言**：人物做了什么比说了什么更重要
3. **留白 > 填满**：70%的故事让读者自己补完
4. **矛盾 > 单一**：最打动人的人物是矛盾的
5. **遗憾 > 完美**：有遗憾的故事比完美结局更让人记住

#### 情感节奏设计

```
铺垫期（平静日常）→ 触发点（小事件）→ 积累期（层层叠加）→ 爆发点（关键时刻）→ 余韵期（回归平静但已不同）
```

**关键：爆发点不能太频繁。** 一季12集，真正催泪的只有2-3集，其余在建立世界观和人物关系。

#### 人物塑造极简公式

1. **一个外部特征**（外貌、习惯、口头禅）
2. **一个内在矛盾**（想要但不敢 / 表面与内心不符）
3. **一个标志性行为**（关键时刻的反应）
4. **一个未说出口的渴望**（驱动行动的真正动力）

**案例参考：**
- 《深夜食堂》茶泡饭三姐妹：只用"又分手了"一句话 + 吃茶泡饭时的表情，让观众自己拼凑出三个人的故事
- 《解忧杂货店》月兔：运动员的坚韧（外部）+ 奥运梦想vs病重恋人（矛盾）+ 反复写信又撕掉（行为）+ 希望有人告诉她该怎么做（渴望）

### 时事融入模块

#### 三层转化法

| 层级 | 方法 | 深度 | 示例 |
|------|------|------|------|
| 第一层 | 直接引用 | 浅 | 角色刷手机看到新闻，皱眉放下 |
| 第二层 | 情境转化 | 中 | 不讨论"AI替代人类"，而是"老板被替换成AI，但AI比真人更体贴" |
| 第三层 | 本质提取 | 深 | 不写"房价太高"，写"寿命可以买卖的城市里穷人只能活到30岁" |

#### 时机选择

- 突发重大事件：2-6小时窗口
- 社会讨论话题：24-48小时
- 文化现象/热梗：1-2周
- 长期社会趋势：不限

#### 三不原则
1. **不生硬**：去掉热点引用故事仍然成立 = 自然融入
2. **不冒犯**：悲剧性事件不适合做讽刺素材
3. **不滞后**：过期热点比没有热点更尴尬

## Step 4: 大纲模板

### 短视频大纲模板（60秒）

```markdown
## [标题]
**风格**：[讽刺/温情/悬疑/混合]
**热点素材**：[如果有]

### 钩子（0-3秒）
[一个让人停下滑动的画面/问题/悬念]

### 建立（3-10秒）
[1-2个细节立住人物 + 快速建立情境]

### 升级（10-40秒）
[冲突逐步升级，至少1个反转]

### 高潮（40-55秒）
[情感/笑点的爆发点]

### 结尾（55-60秒）
[一个让人想分享/评论的画面或金句]
```

### 单元剧大纲模板（5-30分钟）

```markdown
## [标题]
**风格**：[讽刺/温情/悬疑/混合]
**核心概念**：一句话描述（参考《世界奇妙物语》：一个极致的"如果"）

### 出场人物
- [角色名]：[一句话描述 + 一个矛盾]

### 结构
1. **日常开场**（前1/4）：建立正常世界和人物
2. **异常触发**（1/4处）：第一个不正常的事件
3. **升级**（中间1/2）：事态逐步升级，世界观规则被揭示
4. **反转/高潮**（3/4处）：真相揭示或终极反转
5. **余韵**（结尾）：回归日常，但已不同

### 讽刺/情感设计
- **手法**：[选择具体手法]
- **靶点**：[讽刺什么 / 触动什么情感]
- **克制点**：[哪里故意不说透]
```

### 连续剧大纲模板（多集）

```markdown
## [剧名]
**风格**：[讽刺/温情/悬疑/混合]
**集数**：[X]集 × [Y]分钟

### 季主线
[本季要解决的核心冲突]

### 主要角色弧线
- [角色A]：[起点] → [转折] → [终点]
- [角色B]：[起点] → [转折] → [终点]

### 线索分配
- A线（主线）：占比60-70%
- B线（副线）：占比20-30%
- C线（点缀）：占比10%

### 关键节点
- 第1集：[建立世界观 + 触发事件]
- 第5-6集：[季中转折]
- 倒数第2集：[最低谷 / 灵魂黑夜]
- 最后一集：[终极对决 / 终极反转]
```

### 短篇故事模板（500-3000字）

```markdown
## [标题]
**风格**：[讽刺/温情/悬疑/混合]
**字数**：[目标字数]

### 一句话概念
[一个极致的"如果" + 一个矛盾的人物]

### 世界观规则
[这个世界有什么独特规则？用一句话概括]

### 人物
- [主角]：[外部特征] + [内在矛盾] + [未说出口的渴望]

### 结构节拍
1. **钩子**：[第一段：一个让人想继续读下去的场景/句子]
2. **日常**：[主角的"正常状态"，用具体细节建立]
3. **触发**：[打破日常的事件]
4. **升级**：[2-3个逐步加剧的冲突]
5. **反转/高潮**：[核心转折——最好让开头的预期完全落空]
6. **余韵**：[一个具体的细节结尾，不要总结，不要升华]

### 情感/讽刺设计
- **手法**：[具体手法名称]
- **克制点**：[故意留白的地方]
- **参考作品**：[哪部作品的哪种手法]
```

## Step 5: 质量自检清单

完成大纲后，逐项检查：

### 通用检查
- [ ] **意外感**：是否有至少一个读者无法预测的转折？
- [ ] **具体性**：人物是否通过具体的行为/细节立住，而非抽象描述？
- [ ] **矛盾性**：主要人物是否有内在矛盾？
- [ ] **节奏感**：是否有"呼吸"——不是一直在高潮或一直在铺垫？

### 反AI味检查
- [ ] **不工整**：是否存在过于完美的对称结构？能否打破一些？
- [ ] **不直白**：讽刺点是否需要读者自己推理？还是被直接喂到嘴边？
- [ ] **不预告**：笑点/反转是否有"预告感"？能否藏在更自然的叙述中？
- [ ] **有声音**：这个故事是否有"独特的视角"？还是通用的讽刺模板？
- [ ] **有温度**：讽刺中是否带着"善意"？还是纯粹的嘲讽？

### 情感检查
- [ ] **不强行**：是否有直接用情感标签（"他很伤心"、"她很感动"）？
- [ ] **有铺垫**：情感爆发点是否有足够的铺垫？
- [ ] **有余韵**：结局是否留下了回味空间？是否太完美/太说教？

### 时事检查（如适用）
- [ ] **不生硬**：去掉热点引用，故事是否仍然成立？
- [ ] **不滞后**：热点是否仍在时效窗口内？
- [ ] **有人性**：落脚点是否回归到永恒的人性矛盾？
